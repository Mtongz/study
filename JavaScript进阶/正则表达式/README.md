### 1.边界符（位置符） 
* 提示字符所处的位置 ^: start, $: end     同时存在则是精确匹配
### 2. /[abc]/ 表示只要包含a || b || c就true
* /[^abc]/ 括号内^表示取反
### 3.量词符

* 用来设定某个模式出现的次数 /^a{3,6}$/ a可以出现3-6次
```
*:0||more
+:1||more
?:0||1
{n} :n
{n,}:n||more
{n,m}:n-m 
```
#### 案例
##### /^[a-zA-Z0-9_-]{6,16}$/
* 这个模式只能输入英文字母 数字 下划线 短横线
* {6,16} 中间不要有空格

### 4.括号总结：
#### 大括号 量词符 里面表示重复次数
#### 中括号 字符集合 匹配方括号中的任意字符
#### 小括号 表示优先级
